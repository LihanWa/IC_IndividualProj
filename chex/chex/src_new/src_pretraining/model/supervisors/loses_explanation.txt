anat_single 和 anat_multi 分别代表两种不同类型的解剖区域标注方式：
1. anat_single（单框标注）:
每个解剖区域只用一个边界框(bounding box)来标注
例如:用一个框标注"左肺"整体区域
在代码中体现为 target_anat_boxes (形状为 N x A_single x 4)
anat_multi（多框标注）:
每个解剖区域可以用多个边界框来标注
例如:肋骨可能需要多个框来分别标注不同的肋骨
在代码中体现为 target_anat_multiboxes (形状为 N x A_multi x R x 4)
R 表示每个区域最多可以有的框数量
主要区别:
anat_single: 一个解剖区域 = 一个框
anat_multi: 一个解剖区域 = 多个框
这种设计可以更灵活地处理不同类型的解剖结构:
对于整体性的器官(如肺),用单框标注即可
对于分散或多部分的结构(如肋骨、椎骨),可以用多框标注
